# Manual de Usuario

## Registrar un pesaje

Envía una petición POST con el peso del paquete (en kg), el ID de balanza y el ID de paquete. El sistema convierte a Sansas, clasifica y valida las reglas.

## Consultar registros

Puedes obtener los registros filtrando por fecha.

## Estados de un pesaje

Cada registro avanza por estados: INGRESADO → PESADO → APROBADO o RECHAZADO → DESPACHADO.
